# Проект по анализу метрик мобильной игры Zombie Revolution:
## Цель проекта:
Оценить динамику пользовательской базы, активность игроков, финансовые результаты и показатели вирусного роста.
## Методы анализа:
— Изучил исходные данные и провёл анализ количества пользователей и игровых сессий.\
— Рассчитал долю проблемных сессий по типам устройств и оценил возможные причины аномалий.\
— Определил коэффициент вирусности игры и средний размер будущих когорт пользователей.\
— Проанализировал динамику выручки и влияние маркетинговой кампании на финансовые показатели.\
— Оценил изменения ключевых метрик DAU, WAU, MAU и их соотношение.\
— Рассчитал долю лояльных пользователей, протестировав разные критерии определения лояльности.
## Используемые нструменты:
PostgreSQL (CASE WHEN, агрегации, JOIN-ы, CTE, подзапросы), Excel (постобработка, визуализация).
## Результат:
Подготовлена презентация с основными выводами и визуализациями, демонстрирующая динамику пользовательской активности и влияние маркетинговых факторов на выручку игры.
